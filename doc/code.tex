\subsection{Installation}

To install the project's dependencies, navigate to the \texttt{src} directory and optionally, activate a \texttt{virtualenv}. Install with:

\hspace{1cm}\mintinline{bash}{$ pip install -r requirements.txt}

\subsection{Running}

Specify one or more newsfeeds in \texttt{newsgraph.py:36}. Optionally, specify the lists \textit{interests} and \textit{ignore} on the lines below. Run NewsGraph with:

\hspace{1cm}\mintinline{bash}{$ python newsgraph.py}

When the script terminates, the file \texttt{ng.html} in the \texttt{src} directory will contain the metro map and all of its associated data.
